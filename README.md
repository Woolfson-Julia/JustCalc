# JustCalc

A simple, distraction-free calculator.  
Supports: numbers, operators (+, -, *, /), decimal point, and clear function.  
Responsive design: Mobile First, tablet, and desktop.

---

### Project Features

- Mobile First design, adapted for iPhone, tablet, and desktop.  
- Fixed, circular calculator buttons.  
- Formula displayed at the top, current number in large font below.  
- Uses system fonts for fast loading and native appearance.

### `npm start`

Runs the app in development mode.  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.  
The page will reload when you make changes.  
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `build` folder.  
It correctly bundles React in production mode and optimizes the build for the best performance.  

The build is minified and the filenames include the hashes.  
Your app is ready to be deployed!
